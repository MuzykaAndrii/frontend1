Effortless learn

View Source code View Project
About project:

The service intended to simplify learning of English vocabulary. Works as telegram bot, has a web representation for administrative purposes (admin dashboard). After creating vocabulary, you will receive notifications during the day (every 20 mins from 10pm to 22am) with a random word from active vocabulary. The notification will include the sentence example generated by ChatGPT. The philosophy is to memorize lexic by frequently seeing them (also known as Anki method).
